## 使用说明

### 何时更新活跃功能表

- **P5 GC 阶段**：功能完成时，追加到活跃功能表

### 何时移入归档功能表

- 功能被新功能完全替代（使用 `replaced` 状态）
- 功能被下线不再维护（使用 `deprecated` 状态）
- 移动时保留完整的一行记录（不删除），并补充替代信息

### 为什么不删除归档功能

- 历史追溯：了解为什么废弃、是否有类似问题
- 经验参考：新功能设计时可以参考历史实现思路
- 指标连续性：对比 Plan Coverage 等指标的历史趋势
